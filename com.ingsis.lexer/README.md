Com.ingsis.lexer

Propósito:
- Realiza la tokenización del código fuente en tokens mediante múltiples tokenizers (keywords, literals, separators, operators).

Clases clave:
- `Lexer.java`, `DefaultLexer.java` — implementaciones principales.
- `tokenizers/` — tokenizers específicos (keywords, literals, etc.).
- `TokenizersRegistry` — registro/conjunto de tokenizers.

- `src/main/java/com/ingsis/lexer`.
